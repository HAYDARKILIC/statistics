## 📋 About the Course

This repository contains the hands-on Jupyter Notebook materials for the **Statistics for Engineers** course. Each notebook covers the relevant theoretical topics step by step using Python, with a focus on visualisation and support from real-world datasets.

---

## 📁 Notebook Contents

| # | File | Topic | Summary |
|---|-------|------|--------------|
| 01 | `Lecture_01_Introduction_to_Data.ipynb` | **Introduction to Data** | Data matrix, variable types, sampling bias, sampling strategies, random assignment |
| 02 | `Lecture_02_Summarizing_Data.ipynb` | **Summarizing Data** | Histograms, distribution shape, measures of center & spread, box plots, logarithmic transformation, categorical data visualisation |
| 03 | `Lecture_03_Probability.ipynb` | **Probability** | Law of Large Numbers, probability rules, conditional probability, Bayes' Theorem, random variables |
| 04 | `Lecture_04_Distributions.ipynb` | **Probability Distributions** | Normal, Geometric, Binomial, Negative Binomial, and Poisson distributions |
| 05 | `Lecture_05_Statistical_Inference.ipynb` | **Foundations of Statistical Inference** | Point estimates, sampling distribution, Central Limit Theorem, confidence intervals, hypothesis testing, Type I/II errors |
| 06 | `Lecture_06_Inference_for_Categorical_Data.ipynb` | **Inference for Categorical Data** | Single proportion inference, difference of two proportions, Chi-Square goodness-of-fit and independence tests |
| 07 | `Lecture_07_Inference_for_Numerical_Data.ipynb` | **Inference for Numerical Data** | t-distribution, one- & two-sample t-tests, paired data, power analysis, ANOVA, Bonferroni correction |
| 08 | `Lecture_08_Linear_Regression.ipynb` | **Linear Regression** | Least squares, residuals, correlation, types of outliers, inference for regression |
| 09 | `Lecture_09_Multiple_and_Logistic_Regression.ipynb` | **Multiple and Logistic Regression** | Multiple regression, categorical variables, model selection, R², logistic regression, GLM, ROC curve |

## 🛠️ Setup

### Requirements

- Python 3.9 or higher
- Jupyter Notebook / JupyterLab

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HAYDARKILIC/statistics.git
   cd statistics
   ```

2. **Install the required packages:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch Jupyter:**
   ```bash
   jupyter notebook
   ```
